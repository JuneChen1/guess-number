# 猜數字小遊戲(1A2B)
+ 設定 0~9 的四位數字，其中數字不重複
+ 猜題者每次猜四個數字，其中數字不可重複
+ 如果數字和位置都猜對，累計一個 A；數字猜對，但位置不對，累計一個 B
+ 重複猜題，直至猜中 4A 為止
